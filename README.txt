WP Many Posts helps admin to manage blogs with thousands of posts in seconds and save hours of work in bulk actions.

== Description ==

Key feature of WP Many Posts is an advanced inline editing to manage titles, slugs, categories, authors easily and quickly. 
Advanced search and filter to find and edit posts in one second via a powerful Javascript engine and a modern and appealing HTML5 interface . 
PRO version is coming soon with configuration for custom post types and custom fields.
This FREE version is limited to 2500 posts but we can *grant an extension up to 5000 posts upon request at no charge*, more info on http://softrade.it/wp-many-posts-wordpress-plugin

== Installation ==

1. Download and extract `wp-many-posts.zip` to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the post grid directly from the main menu trough the "Wp Many Post" link
4. Configuration parameters are available under the "Wp Many Post => Settings" submenu.
5. For more information of this plugin, please visit: [Plugin Homepage](http://softrade.it/wp-many-posts-wordpress-plugin "Wp Many Posts").


== Usage ==
From the grid you can filter post by Title, Author, Slug and status.
You can also group posts by Status and author dragging the column header in the main header.
Clicking on "edit" will poen a popup window from  where you can make your changes to alla the available parameters plus add /remove tags and categories
Clicking on "open" you'll open a new window in the standard edit screen for the single post
